网页知识库导入

输入网址自动爬取网页内容并导入知识库

最大递归深度
1 2 3 4 5

深度 1: 仅当前页面
深度 2-3: 推荐，爬取子页面
深度 4-5: 完整站点爬取

使用说明

  • 系统会自动爬取起始网址及其所有子链接
  • 递归深度最大为 5 层
  • 每个网页内容会被分割成片段并生成向量嵌入
  • 自动检测重复内容，避免重复导入
  • 可在「导入任务」页面查看进度

视频知识库导入

下载视频并提取语音转换为文本导入知识库

视频处理流程

  • 从 URL 下载视频或使用本地视频文件
  • 提取视频中的音频轨道
  • 使用 Whisper AI 进行语音识别
  • 将识别的文本按时间戳分割
  • 生成向量嵌入并导入知识库
  • 支持按时间戳定位视频片段
